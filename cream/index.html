What?

Cream is an events library written in pure javascript, when minified it falls in at 0.5kb.

Browser Support

Note: this is only the browsers I've tested it in, it probably works in a whole load more than this.

  • IE8+
  • Firefox 21+
  • Chrome 30+

Usage

The following code is what is used for the examples below

var click = Cream('#click').on(['click', 'sample'], [function(ev) { // log: click }, function() { // log: fired sample var buttonSample = this; buttonSample.innerHTML = 'Boom'; setTimeout(function() { buttonSample.innerHTML = 'Click'; }, 2000); }]); Cream('#hover').on(['mouseenter', 'mouseout'], [function(ev) { // log mouseenter }, function(ev) { // log mouseleave }]); Cream('#trigger').on(['click'], [function(ev) { click.trigger('sample'); }]);
